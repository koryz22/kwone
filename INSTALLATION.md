# Installation Guide

## Quick Start

Your Shopify theme is ready! The ZIP file `kwone-lookbook-theme.zip` has been created and is ready to upload.

## Upload to Shopify

### Step 1: Access Shopify Admin
1. Log in to your Shopify admin panel
2. Navigate to **Online Store** → **Themes**

### Step 2: Upload Theme
1. Scroll to the bottom of the Themes page
2. Click **"Add theme"** button
3. Select **"Upload zip file"**
4. Choose `kwone-lookbook-theme.zip` from your computer
5. Wait for the upload to complete (usually 30-60 seconds)

### Step 3: Preview & Customize
1. Once uploaded, your theme will appear in the "Theme library" section
2. Click **"Actions"** → **"Preview"** to see how it looks
3. Click **"Customize"** to adjust settings:
   - Brand name and tagline
   - Collection selection
   - Layout style (Editorial or Minimal Grid)
   - Colors and typography

### Step 4: Publish
1. When you're happy with the preview, click **"Actions"** → **"Publish"**
2. Confirm the publish action
3. Your lookbook is now live!

## Customization Tips

### Lookbook Section Settings
- **Brand Letter**: Single letter displayed in top-left corner (e.g., "K")
- **Brand Name**: Your brand name (defaults to shop name)
- **Tagline**: Descriptive text below brand name
- **Collection**: Select which products to show (leave empty for all products)
- **Layout Style**: 
  - **Editorial**: Notebook/archive aesthetic with varied product placement
  - **Minimal Grid**: Clean, uniform grid layout

### Theme Settings
- **Typography**: Choose body and heading fonts
- **Colors**: Customize text, background, and accent colors
- **Favicon**: Upload a 32x32px icon for browser tabs

## Troubleshooting

### Theme won't upload
- Ensure the ZIP file is not corrupted
- Check file size (should be under 50MB)
- Try re-running `create-theme-zip.ps1` to regenerate the ZIP

### Products not showing
- Make sure you have products in your store
- Check the collection selection in theme settings
- Verify products are published and available

### Layout looks broken
- Clear browser cache
- Check that all theme files uploaded correctly
- Try previewing in a different browser
